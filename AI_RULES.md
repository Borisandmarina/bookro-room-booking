# Codex Rules for this project

When modifying code in this repository, follow these strict rules:

- Do NOT rewrite entire files
- Do NOT change formatting
- Do NOT rename functions, variables, hooks, or database fields
- Do NOT modify unrelated code
- Do NOT refactor or optimize
- Do NOT add new features

Only apply minimal, targeted changes required for the task.

Before making changes:
1. Identify exact locations in code
2. Show what lines will be modified

After changes:
- Show a clear diff (before → after)
- Limit changes to the smallest possible scope

If more than 10 lines would be changed:
- STOP and show analysis only

Priority:
Stability and minimal risk over code quality improvements.
